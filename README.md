# ech-braze
A WordPress plugin to track Braze custom events (Lead Forms & WhatsApp).

## Usage
Backend Configuration
After activating the plugin, navigate to ECH Braze in your WordPress dashboard to provide the following required credentials:

Braze API Key: Your Web SDK API Identifier.

Braze Base URL: The SDK endpoint for your cluster (e.g., sdk.braze.com or sdk.braze.eu).

Business Name: The identifier for the current site (e.g., DR REBORN).
